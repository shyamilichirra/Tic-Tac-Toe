# Tic-Tac-Toe

A simple implementation of the classic Tic-Tac-Toe game.

## Features
- Play against a friend (2-player mode)
- Basic game logic with win/draw detection
- Simple UI

## Usage
1. Clone the repository.
2. Open `http://127.0.0.1:5500` in your browser.
3. Play the game!

## Example
- Player X places mark
- Player O places mark
- Game declares winner or draw
